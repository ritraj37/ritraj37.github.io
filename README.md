# 🌐 Personal Portfolio Website

This is my **personal portfolio website** showcasing my projects, skills, and experience as a developer. The portfolio highlights various **web development projects** built using HTML, CSS, and JavaScript.

The goal of this portfolio is to present my work, demonstrate my technical skills, and provide an easy way for recruiters and collaborators to explore my projects.

---

## 🚀 Features

* Responsive portfolio design
* About Me section
* Skills showcase
* Projects section with mini projects
* Resume section
* Contact section
* Navigation bar and footer
* Organized project structure

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and responsive layout
* **JavaScript** – Interactivity and dynamic behavior
